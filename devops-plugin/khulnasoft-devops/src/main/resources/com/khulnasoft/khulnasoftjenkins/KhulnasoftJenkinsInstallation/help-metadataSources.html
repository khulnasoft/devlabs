Specify the metadata source name for different event types. This allows you to customize how events from different parts of your Jenkins build process are categorized and routed in Khulnasoft.

Common event types include:

  • test: For JUnit test reports
  • coverage: For code coverage reports
  • notifications: For build notifications and alerts
  • build_metrics: For build performance and metrics data

If no specific source is configured for an event type, the default source will be used.
